+ ## Series
  + ## 範例
    ```python
    import numpy as np
    from pandas import Series

    a = [1, 2, 3, 4]

    # 創建一個Series
    # s = Series(a)
    # print(s)
    # print(s.index)
    # print(s.values)

    # 自定義index
    # s1 = Series(a, index=['A', 'B', 'C', 'D'])
    # print(s1)

    # 透過numpy的array創建Series
    # s2 = Series(np.arange(5))
    # print(s2)

    # 透過字典創建Series
    d = {
        'A': 1,
        'B': 2,
        'C': 3,
        'D': 4
    }
    s3 = Series(d)
    # print(s3)
    # print(s3.index)
    # print(s3.values)

    # 將Series轉換成字典
    d2 = s3.to_dict()
    print(d2)


    """
    index會默認自動創建，也可以指定
    用字典創建的Series，字典中的key會變成index，而value就會變成value
    將Series轉成字典的話，就是index轉乘key，value轉成value
    """
    ```
