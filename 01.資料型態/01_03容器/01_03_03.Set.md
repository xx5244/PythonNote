# Set
---

+ ## 定義
  1. 無序的，可變的，不重複的容器
  1. 用{ }來包住數據
  1. 無序，所以沒有索引取值
  1. 資料型態為set
  1. 元素若要放容器類只能放Tuple，其它皆不行
  1. 重複的元素會被刪掉
    
  + ### 範例
    ```python
    a = {1, 2, 3}
    b = {"a", "b", "c"}
    c = {True, False, True}
    d = {None}
    e = {}

    print(a)
    print(b)
    print(c)
    print(d)
    print(e)
    ```

+ ## 長度
  ### 用len指令可以查詢List元素

  ```python
    a = {1, 2, 3, 1}
    print(len(a))
  ```

+ ## 應用
  + ### 去除List重複值
  
    ```python
    li = [7, 2, 2, 8, 3, 5, 1, 6, 4, 4, 8, 1, 7, 7]
    li = list(set(li))
    print(li)
    li.sort(reverse=True)   # 若要降序的話
    print(li)
    ```
