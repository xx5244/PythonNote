# Tuple
---

+ ## 定義
  1. 有序，不可變的容器
  1. 用( )來包住數據，或者不加任何( )最後用`，`來結尾
  1. 數據之間用`，`隔開
  1. 索引用[ ]來取值，由0開始
  1. 資料型態為tuple
  1. 注意，如果是用( )方法表示的話，若( )內部資料只有單筆，那資料型態則會為該筆資料的資料型態而不會是Tuple

  + ### 範例
    ```python
    # 可放各種數據，數字、文字、布林數、各種容器
    a = 1, 2, 3,    # 這種表示方法也可以
    b = ("a", "b", "c")
    c = (True, False, True)
    d = (None)
    e = ()
    f = a, b, c, d, e,  # 這種表示方法也可以
    g = "123",  # 資料型態還會是Tuple
    h = ("123")  # 資料型態會是str

    print(a)
    print(a[0])
    print(b)
    print(b[-1])
    print(c)
    print(d)
    print(e)
    print(f)
    print(type(g))
    print(type(h))
    ```

+ ## 優點
  1. 速度比List快
  2. 對不需改變的數據做寫入保護的機制，使得code更安全
  3. Tuple可以做為dict的Key value，也可做為Set的元素，而List則否
 
+ ## 串接
    ### 用' + '即可把Tuple串接

    ```python
    a = 123, 456,
    b = ("abc", "def")
    c = (True, False)
    print(a + b + c) # 當然還是建議，不同資料型態不要擺在同個list中
    ```
+ ## 重複
    ### 用' * '即可把Tuple的內容重複N次
    ```python
    a = 123, 456,
    b = "abc", "def",
    c = (True, False)
    print(a * 4)
    print(b * 4)
    print(c * 4)
    ```
+ ## 長度
    ### 用len指令可以查詢Tuple元素
    ```python
    a = "abc", "def",
    print(len(a))
    ```
