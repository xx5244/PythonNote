# 數據持久化(Data Persistence)
---

+ ## 定義
    **儲存在磁盤上不太會被改變的數據**

+ ## 序列化
  + **序列化是將數據結構或對象狀態儲存起來的過程**
  + ### 序列化儲存3步驟
    1. **打開文件**
    2. **pickle.dump(obj, file)**
    3. **關閉文件**
  + ### 範例
    ```python
    class People:
            def __init__(self, name, age):
                self.name = name
                self.age = age
            def sayhi(self):
                print(f'Hi I\'m {self.name}, I\'m {self.age} years old')

        # p = People(age=18, name='測試者')
    ```

  + ### 序列化對象加載3步驟
