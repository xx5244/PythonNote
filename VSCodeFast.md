# VS Code 常用快捷鍵
---

+ ### ctrl + shift + k
    ### 刪除整行程式碼

+ ### shift + alt + a
    ### 整塊區塊內容變成字串

+ ### alt + 上/下鍵
    ### 上/下移動整行程式碼

+ ### alt + shift + 上/下鍵
    ### 複製整行程式碼放置上/下一行

+  ### alt + 滑鼠左鍵
    ### 多鼠標

+ ### ctrl + alt + 上/下鍵
    ### 上/下多鼠標

+ ### ctrl + shift + p
    ### 打開命令列

+ ### ctrl + shift + n
    ### 打開新的VS Code視窗
