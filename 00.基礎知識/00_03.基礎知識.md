+ ## 註解
註解用`#`來開頭，可完成單行註解，或按ctrl + /
多行註解方式有2種

  1. 可用滑鼠框選想註解的區塊，然後按ctrl + / 一次註解多行
  1. 用欲註解區塊前後分別用3個雙引號來包住

```python
    # 單行註解

    """
    多行註解
    很多行
    """        
```

+ ## 命名
命名有大小寫之分
```python
a = 123
A = 456
# A 不等於 a
```

+ ## 索引
索引用[ ]來取，各容器要取值都能用索引的方式
python中索引值都是從0開始
```python
a = 1, 2, 3
print(a[0])
```
